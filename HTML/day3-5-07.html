Favicon

A favicon is a graphic image (icon) associated with a particular website or webpage.

How to add Favicon?

  1. Place the image in the root directory of the website
  2. Add the following code in the head section of the HTML document
  3. link rel="icon" type="image/x-icon" href="image_path/url"
