The taget attribute can have one of the following values:

  1. _self: Default, open the document or link in a same window or tab as it is clicked
  2. _blank: Opens the document or url in a new tab or window
  3. _parent: Opens the document in the parent frame
  4. _top: Open the document in the full body of the window
