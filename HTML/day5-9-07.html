HTML Head

Html head is like a element, in this we can include title, style, meta, link, script

HTML Head element

1. head: Defines the information about the document
2. title: Defines the title of the document
3. link: Defines the relationship between the current document and an external resource
4. style: Defines the style information for the document to visually look good
